## 🧩 Description
**Webitrust** est une extension navigateur qui évalue la **fiabilité et la réputation des sites web** visités.  
Elle s’adresse à tous les utilisateurs, même non techniques, pour les aider à identifier les sites potentiellement malveillants.

L’objectif est de rendre la navigation plus sûre en combinant **analyse automatisée** et **contribution communautaire**.

---

## 🚀 Fonctionnalités principales

- Analyse automatique d’un site :
  - Âge du domaine
  - Réputation IP
  - Vérification des scripts suspects
  - Références croisées avec des bases de données de sécurité (ex: Fortinet, etc.)
- Attribution d’un **score de fiabilité** pour chaque site
- Système de **vote utilisateur** pour affiner la réputation
- Page d’alerte et blocage sur demande d’un administrateur
- Tableau de bord administrateur avec statistiques globales
- Interface simple, fluide et intuitive

---

## 🎯 Objectifs du projet

- Développer une **extension fonctionnelle en version bêta** testée par des utilisateurs externes.  
- Fournir une expérience fluide, stable et compréhensible par tous.  
- Calculer des scores de fiabilité sur plusieurs centaines de sites.  
- Rendre l’extension publiquement accessible et utilisée en dehors de l’équipe.

---

## 🧠 Compétences mobilisées

- Développement web (HTML, CSS, JavaScript)
